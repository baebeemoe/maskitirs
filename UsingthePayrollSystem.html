3. Using The Payroll System


This section is designed to guide users through the various functionalities and processes involved in effectively managing payroll tasks within the system. Whether the user is responsible for processing payroll, managing employee records, generating reports, or overseeing administrative tasks, this section provides step-by-step instructions and best practices to help the user navigate the payroll system with confidence.


3.1 Secure Login Procedures


Employees will be provided with an employee number and an initial password, also the ability to recover/reset passwords as needed. Role-based access control (RBAC) implementation is pertinent, limiting user access to the required features and information per their job roles and authorization. The password resets every month. Below are the login credentials that each actor must have in order to be authenticated.


Actors Authentications
Employee Login: username and password
HR Department Login: username and password
Finance Department Login: username and password
IT Department Login: username and password

3.1.1 To log into the payroll system:


  1. Open the MotorPH Payroll System desktop application.
  2. Enter Employee ID and Password.
  3. Click Login.

3.3 Viewing the Profile Information


Employees utilizing the payroll system has the ability to view their information through the profile page after successfully logging in.


3.3.1 To view the profile information:


  1. Click Profile on the left sidebar menu.
  2. The profile panel will display employee information and provide an option for the user to change their password.


3.4 Changing the Password


Employees are encouraged to immediately updated their password on the first login and to regularly make changes to ensure account security. The ability to change the password is available on the profile page.


3.4.1 To change the user password after login:


  1. Click Profile on the left sidebar menu.
  2. Click the Change Password button.
  3. Enter the new password and enter it again to confirm in the Change Password dialog box.
  4. Click the Update Password button to save the changes.


3.5 Viewing the Attendance Records


The payroll system provides a platform for the employees to view their attendance records, including overtime hours incurred. The attendance records can be viewed in the Attendance page


3.5.1 To view the attendance records after login:


  1. Click Attendance on the left sidebar menu.
  2. The attendance panel will display employee attendance records showing dates, timestamps of time in and time out, and overtime hours.


3.6 Viewing the Overtime Request Records


Overtime requests are recorded by the system and displayed in a dedicated panel under the Attendance page.


3.6.1 To view the overtime request records after login:


  1. Click Attendance on the left sidebar menu.
  2. Click the Overtime Requests tab to view the list of the overtime requests initiated by the employee.
  3. The overtime requests table includes the date, start time, end time, and the status of the overtime request.


3.7 Requesting Overtime Work


Within the payroll system, all employees have the ability to request approval for overtime work. This feature can be accessed in the Attendance page, in the Overtime Request panel.


3.7.1 To request approval for overtime work:


  1. Click Attendance on the left sidebar.
  2. Click Overtime Request tab.
  3. Click the Request Overtime button.
  4. Enter the Date when overtime work happens.
  5. Enter the Start Time and End Time.
  6. Enter the Reason for the overtime work.
  7. Click Submit.
  8. The overtime request will be displayed on the overtime request table with a Pending status as the employee awaits approval.


3.8 Leave Applications


The payroll system provides a channel where the employees can submit their leave requests conveniently. The leave application can be accessed in the Leave page. The type of leave of absences in this application are: Vacation Leave, Sick Leave, and Emergency Leave.


3.8.1 To apply for leave:


  1. Click Leave on the left sidebar.
  2. Click the Request Leave button.
  3. Select the type of leave from the Leave Type dropdown
  4. Enter the Start Date and the End Date of the leave.
  5. Enter the Reason for the leave.
  6. Click Submit.
  7. The leave request will be displayed on the leave request table with a Pending status as the employee awaits approval.


3.9 Viewing the Leave Request Records


Employees have the ability to view their leave request records to monitor the status of the request. The Leave Request History can be accessed in the Leave page.


3.9.1 To view the leave request records:


  1. Click Leave on the left sidebar.
  2. The Leave Request History table is automatically populated with the leave request information including the date the request was filed, type of leave, start date, end date, reason, and status.


3.10 Viewing the Payslip


The payroll system offers a platform for the employees to view the details of their income, along with the breakdown of their earnings, benefits, deductions, and tax amount.


3.10.1 To view the payslip:


  1. Click Payslip on the left sidebar.
  2. Select the Cut-off Dates from the dropdown menu.
  3. Click the View Payslip button to view the payslip generated for the selected cut-off dates.
  4. To view payslips from other cutoff dates, repeat Step 2 and 3.


3.11 Approving/Denying Leave Requests


The ability to approve or deny a leave request is accessible only to authorized users. Supervisors are authorized to approve leave requests. Changes to the status of the request are updated real-time.


3.11.1 To approve a leave request after successful login as a supervisor:


  1. Click the Manage Requests button on the dashboard.
  2. Leave Requests are displayed first by default on the Requests screen.
  3. Locate the Pending request.
  4. Click the status of the request.
  5. Select Approved from the dropdown.
  6. Click the Save button at the bottom of the screen.
  7. Select Yes to confirm the approval.


3.11.2 To deny a leave request after successful login as a supervisor:


  1. Click the Manage Requests button on the dashboard
  2. Leave Requests are displayed first by default on the Requests screen.
  3. Locate the Pending request.
  4. Click the status of the request.
  5. Select Denied from the dropdown.
  6. Click the Save button at the bottom of the screen.
  7. Select Yes to confirm the approval.


3.12 Approving/Denying Overtime Requests


Only authorized users, such as the supervisor, are allowed to approve or deny overtime requests. Changes to the status of the request are updated real-time.


3.12.1 To approve an overtime request after successful login as a supervisor:


  1. Click the Manage Requests button on the dashboard.
  2. Click the Overtime Requests button to access the overtime request list.
  3. Locate the Pending request.
  4. Click the status of the request.
  5. Select Approved from the dropdown.
  6. Click the Save button at the bottom of the screen.
  7. Select Yes to confirm the approval.


3.12.2 To deny a leave request after successful login as a supervisor:


  1. Click the Manage Requests button on the dashboard.
  2. Click the Overtime Requests button to access the overtime request list.
  3. Locate the Pending request.
  4. Click the status of the request.
  5. Select Denied from the dropdown.
  6. Click the Save button at the bottom of the screen.
  7. Select Yes to confirm the approval.


3.13 Managing Employee Information


Authorized users have the ability to manage employee information through the payroll system. HR Administrators can access the Manage Employee Information feature through the dashboard. Managing employee information includes adding, updating, and deleting individual employee information, and also importing new employee information in bulk.


3.13.1 Accessing the Manage Employee Information Screen


  1. Click the Manage Employee Information button on the dashboard.
  2. The Manage Employee Information screen is composed of an employee search bar, employee table, and the menu on the left side.


3.13.2 Employee Search


  1. Click the search bar at the top of the screen.
  2. Type an employee information, e.g., first name or last name.
  3. Click the Search button to initiate search.
  4. The employee table will be updated with rows that include the information typed on the search bar.


3.13.3 Adding New Employee Information


  1. Click the Add Employee button on the menu.
  2. Enter all required information in the Add Employee form.
  3. Click the OK button to add the new employee information.
  4. The employee table will be updated with the newly added information.


3.13.4 Updating Employee Information


  1. Select an employee from the employee table.
  2. Click the Update Employee button.
  3. Modify the fields that need to be updated.
  4. Click the OK button to save the changes.
  5. The employee table will be updated with the newly updated information.


3.13.5 Deleting Employee Information


  1. Select an employee from the employee table.
  2. Click the Delete Employee button on the menu.
  3. Confirm to delete the employee information.


3.13.6 Viewing Employee Information


  1. Select an employee from the employee table.
  2. Click the Delete Employee button on the menu.
  3. View the employee information.


3.13.7 Bulk Import of Employee Information


  1. Click the Import CSV on the menu.
  2. Locate the CSV file that contains employee data.
  3. Click the Open button on the file explorer.
  4. Confirm or cancel import after the system checks for duplicates.
  5. The employee table will be updated with the newly added employee information.


3.13 Monthly Payroll Report


The payroll system automatically generates monthly payroll reports, and these reports are only accessible to payroll administrators.


3.14.1 To access the Monthly Payroll Report after successful login as a payroll administrator:


  1. Click the Manage Payroll button on the dashboard.
  2. Click the Monthly Payroll Reports tab.
  3. Select the month from the dropdown list.
  4. The monthly payroll report table will be automatically populated.
