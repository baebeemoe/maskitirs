2. Getting Started


Before diving into the functionality of the MotorPH Payroll System, it's essential to ensure that the hardware and software meet the necessary requirements and that the user has the appropriate access to begin using the system. This section provides an overview of the hardware prerequisites, software prerequisites, and instructions for accessing the payroll system


Ensure that the userâ€™s system meets these requirements before installing the MotorPH Payroll System. Failure to meet these requirements may result in compatibility issues or suboptimal performance.


2.1 Hardware Prerequisites


The following are the minimum hardware requirements to run the payroll system as expected.


  • Processor: Intel Core i3 processor or equivalent
  • Memory (RAM): 8 GB RAM or higher
  • Storage: 100 GB of available hard disk space
  • Display: Monitor with a minimum resolution of 1280x800 pixels

2.2 Software Prerequisites


The following are the minimum software requirements to run the payroll system as expected.


  • Operating System: Windows 10 (64-bit), macOS 10.14 or later, or a compatible Linux distribution
  • Java Runtime Environment (JRE): Oracle JDK 8 or later

2.3 Additional Requirements


  • Administrator Privileges: Users installing the Payroll System must have administrative privileges on their computer
  • Security Software: Ensure that firewall and antivirus software are configured to allow the Payroll System to function properly
  • Backup Solution: Implement a regular backup solution to protect payroll data from loss or corruption.

2.4 Accessing The System


The payroll system can be accessed through a dedicated application. The conventional method entails installing the software on a business's servers. The system executes authorization after verifying a user's identity. This procedure establishes a user's access level to the payroll system. Permissions may differ amongst user roles (e.g., employee, HR, IT, Payroll).


  1. Download the payroll system installer here.
  2. Click and run the downloaded application.
  3. Allow the necessary system permissions (if applicable).
  4. Log in with security credentials.
